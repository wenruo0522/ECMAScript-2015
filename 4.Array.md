# Array

### 1. 遍历数组

- for 循环

  ```js
  let arr = [1, 2, 3, 4]
  ```

  

- forEach(): 没有返回值，对每个元素调用 func

- map(): 返回新的Array，每个元素为调用 func 的结果

- filter(): 返回符合 func 条件的元素数组

- some(): 返回 boolean, 判断是否有元素符合 func 条件

- every(): 返回 boolean, 判断每个元素是否符合 func 条件

- reduce(): 接受一个函数作为累加器

- for ... in ...
