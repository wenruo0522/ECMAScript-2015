# Object

### 1. 对象的扩展

- 属性简洁表示法

  ```js
  let name = 'jack'
  let age = 20
  let obj = {
      name,
      age,
      study() {
          console.log(this.name)
      }
  }
  ```
  
- 属性名表达式

  ```js
  let name = 'jack'
  let s = 'school'
  let obj = {
      name,
      [s]: 'immoc'
  }
  console.log(obj) // { name: "jack", school: "immoc" }
  ```
  
- Object.is

  判断两个对象是否相等

  ```js
  let obj1 = {
      name: 'jack',
      age: 20
  }
  
  let obj2 = obj1
  ```
